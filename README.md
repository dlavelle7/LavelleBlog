# LavelleBlog #
Blogging web app I wrote in June 2013

## Live Site ##
http://lavelle-blog.herokuapp.com/login

## Local Installation ##
* pip install -r requirements.txt
* mkdir tmp
* touch tmp/lavelleblog.log
* python run.py
